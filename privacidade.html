Política de Privacidade

MultiplicaMente — Aventura da Tabuada

📋 Visão Geral

O MultiplicaMente é um aplicativo educacional de matemática desenvolvido para crianças. Levamos a privacidade dos nossos usuários, especialmente de menores de idade, muito a sério.

Este aplicativo é destinado a crianças a partir de 6 anos e foi desenvolvido em conformidade com a Lei Geral de Proteção de Dados (LGPD) e o Marco Civil da Internet.

👤 Dados Coletados

O MultiplicaMente coleta apenas os dados estritamente necessários para o funcionamento do aplicativo:

  • Nome do jogador — definido pelos pais/responsáveis no momento da criação do perfil
  • Progresso no jogo — estrelas, gemas, criaturas desbloqueadas, histórico de partidas
  • E-mail e conta Google — apenas quando o responsável opta por criar uma conta para sincronização na nuvem (opcional)
⚠️ Não coletamos dados de localização, contatos, câmera, microfone ou qualquer informação sensível.

🔒 Como Usamos os Dados

  • Salvar o progresso do jogador no dispositivo (armazenamento local)
  • Sincronizar o progresso na nuvem quando o responsável cria uma conta (opcional)
  • Exibir estatísticas de desempenho para os pais/responsáveis

Não utilizamos os dados para publicidade, não vendemos dados a terceiros e não compartilhamos informações com anunciantes.

☁️ Armazenamento e Segurança

  • Os dados locais são armazenados apenas no dispositivo do usuário
  • Quando a sincronização na nuvem é ativada, os dados são armazenados de forma segura no Google Firebase, com autenticação obrigatória
  • Cada usuário acessa apenas seus próprios dados — não há acesso cruzado entre contas
  • Utilizamos criptografia padrão do setor para proteger todas as transmissões de dados

👨‍👩‍👧 Proteção de Menores

O MultiplicaMente foi desenvolvido especificamente para uso infantil sob supervisão de pais e responsáveis:

  • Não há comunicação entre usuários (sem chat, sem fórum)
  • Não há publicidade de terceiros no aplicativo
  • Não há compras realizadas pela criança sem ação do responsável
  • O PIN dos pais protege o acesso ao painel de desempenho e configurações
  • Compras premium (in-app purchase) requerem confirmação do responsável via Google Play

🗑️ Exclusão de Dados

Os responsáveis podem a qualquer momento:

  • Apagar o perfil da criança diretamente no aplicativo
  • Solicitar a exclusão completa dos dados da nuvem pelo e-mail abaixo
  • Desinstalar o aplicativo para remover todos os dados locais
